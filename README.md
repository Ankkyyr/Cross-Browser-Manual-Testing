# Cross-Browser Manual Testing & QA Documentation

## Project Overview
This repository contains comprehensive manual testing documentation, test case executions, and browser compatibility reports for an E-Commerce web application. The testing strategy ensures a seamless user experience across multiple environments and browsers.

## Key Achievements
* **Cross-Browser Testing:** Tested the web application across Chrome, Firefox, Edge, and Safari browsers.
* **Test Execution:** Executed 30+ manual UI, usability, compatibility, and exploratory testing scenarios.
* **Documentation:** Maintained detailed test case execution logs and comprehensive browser compatibility reports to assist developers in debugging UI inconsistencies.

## Repository Structure
* `Test_Execution_Report.csv` - Detailed manual test cases, scenarios, and execution status.
* `Browser_Compatibility_Matrix.csv` - Cross-browser testing results for core application features.
